# Вопросы по БЗ ЦК CV

## Организация

* Вопрос: Чем занимается Центр компетенций по Computer Vision? За что отвечает?
> Ответ: Центр Компетенций по Computer Vision занимается развитием навыков и компетенций сотрудников Службы Внутреннего Аудита в области компьютерного зрения. Основные направления деятельности ЦК включают:
>- Совершенствование методологии и повышение квалификации сотрудников по направлениям CV.
>- Накопление базы знаний и обмен опытом между специалистами.
>- Разработка собственных моделей и решений в сфере компьютерного зрения.
>- Разработка инструментов для упрощения работы в области CV.
>- Привлечение подразделений Службы Внутреннего Аудита (УВА) к достижению поставленных целей.
>
>Также ЦК предлагает различные курсы повышения квалификации в области компьютерного зрения. Таким образом, Центр Компетенций играет ключевую роль в развитии технологий компьютерного зрения, обеспечивая внедрение современных методов анализа визуальной информации для улучшения качества работы и эффективности внутренних проверок.

* Вопрос: Какие собственные разработки есть в ЦК на текущий момент
 Ответ:

На текущий момент в ЦК по СV есть модели решающие следующие задачи:
Детекция сотрудник/клиент, детекция рабочего стола сотрудника ВСП, алгоритм определения 2-х сотрудников в рабочей зоне.
Подсчет количества людей на изображениях, определения уровня освещенности помещения.
Распознавание подписей на документах.
Детекция маски на лицах.
Поиск лиц в паспортах, определение схожести фото в разных паспортах.
Определение сотрудников, спящих на рабочем месте, определение отсутствия сотрудников на рабочем месте.
Определения ношения сотрудниками ЧОП рабочей формы.
Определение использования телефона на рабочем месте.


А так же инструмент работающий со SmartView для исследования доступных камер, заказа снэпшотов и заказа видео из архива.
