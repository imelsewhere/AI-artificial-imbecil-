# Глоссарий по БЗ ЦК по Компьютерному зрению

## Компьютерное зрение

* CV
 Computer Vision (Компьютерное зрение). Область искусственного интеллекта, которая позволяет компьютерам "видеть", понимать и анализировать содержимое изображений и видео.


* ML
 Machine Learning (Машинное обучение). Технология, которая позволяет компьютерам учиться на данных и принимать решения без явного программирования для каждой конкретной задачи.


* DL
 Deep Learning (Глубокое обучение). Подраздел машинного обучения, а котором используются искусственные неронные сети с множеством слоёв.


* Classification (Классификация)
 Отнесение всего изображения к одному классу (например, "кошка", "собака", "человек").


* Bounding Box (BBox)
 Ограничивающий прямоугольник вокруг объекта.


* Object Detection (Обнаружение объектов)
 Нахождение объектов на изображении и их локализация с помощью Bounding Box.


* OCR
 Optical Character Recognition (Оптическое распознавание символов). Преобразование текста с изображений в машиночитаемый формат.


* CNN
 Convolutional Neural Network (Сверточная нейронная сеть). Это спеуиальный тип нейронной сети, созданный для эффективной работы с изображениями. Это архитектура нейросети, которая видит изображение по частям и учится распознавать образы от простых к сложным.


* Transformer
 Архитектура нейроной сети, которая обрабтывает анные целыми последовательностями с помощью механизма внимания, вместо анализа по порядку.


* ResNet
 Глубокая сверточная нейронная сеть, которая решает ключевую проблему очень глубоких сетей - проблему исчезающего градиента. Использует skip conections, чтобы успешно обучать очень глубокие нейросети.


* YOLO (You Only Look Once)
 Система обнаружения объектов, разработанная для быстрого и точного обнаружения объектов на изображениях или видео. Особенность - однократный анализ всего изображения за один проход.


* Augmentation (Аугментация)
 Искусственное увеличение размера датасета путем случайных преобразований исходных изображений (повороты, отражения, изменение цвета и т.д.).


* Transfer Learning (Трансферное обучение)
 Техника, при которой модель, предварительно обученная на большом датасете (например, ImageNet), дообучается на своей, более специфической задаче.


* Свертка
 Операция, когда небольшая матрица-фильтр "скользит" по изображению и выделяет в не определенные особенности. Это способ научить компьютер видеть в изображении важные черты - края, текстуры, узоры.
